# 🚀 Graph Questions in Java

> A curated collection of Graph Data Structure and Algorithm problems solved in Java for coding interviews, placements, and competitive programming.

---

## 📌 About

This repository contains Java implementations of important Graph algorithms and interview questions. It is designed to help students strengthen their understanding of graph concepts and prepare for technical interviews.

---

## 🎯 Objectives

- Learn Graph fundamentals
- Master graph traversal algorithms
- Solve interview-focused problems
- Improve problem-solving skills
- Prepare for placements and coding contests

---

## 📚 Topics Covered

### 🔹 Graph Basics
- Graph Representation
- Adjacency Matrix
- Adjacency List

### 🔹 Graph Traversal
- Breadth First Search (BFS)
- Depth First Search (DFS)

### 🔹 Cycle Detection
- Undirected Graph (BFS)
- Undirected Graph (DFS)
- Directed Graph (DFS)
- Kahn's Algorithm

### 🔹 Topological Sorting
- DFS
- Kahn's Algorithm

### 🔹 Shortest Path Algorithms
- BFS (Unweighted Graph)
- Dijkstra's Algorithm
- Bellman-Ford Algorithm
- Floyd-Warshall Algorithm

### 🔹 Minimum Spanning Tree
- Prim's Algorithm
- Kruskal's Algorithm

### 🔹 Disjoint Set (Union-Find)

### 🔹 Strongly Connected Components
- Kosaraju Algorithm
- Tarjan Algorithm

### 🔹 Other Important Problems
- Bipartite Graph
- Number of Islands
- Rotten Oranges
- Word Ladder
- Alien Dictionary
- Bridges
- Articulation Points

## ⏱️ Time Complexity

| Algorithm | Time Complexity |
|-----------|-----------------|
| BFS | O(V + E) |
| DFS | O(V + E) |
| Dijkstra | O(E log V) |
| Bellman-Ford | O(V × E) |
| Floyd-Warshall | O(V³) |
| Prim's | O(E log V) |
| Kruskal's | O(E log E) |
